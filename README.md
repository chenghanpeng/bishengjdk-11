## Boole JDK 11

Boole JDK 是一个高性能、多平台支持、可用于生产环境的OpenJDK发行版。

二进制可以从[这里](https://gitee.com/openeuler/boolejdk-11/releases)下载。

## 平台支持

Boole JDK 当前同时支持 `Linux/x86_64` 和 `Linux/AArch64` 两种平台。
